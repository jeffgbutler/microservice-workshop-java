Plan for the Workshop

We will use Spring Boot for every example and exercise. Major topics include:

  • Building Basic Web Services
  • Building Web Service Clients with Feign (Option 1)
  • Building Web Service Clients with REST Template (Option 2)
  • Using Resilience4j for Circuit Breakers
  • Using Eureka for Service Discovery

Caveats

  • This workshop is focused on learning some fundamental tools and patterns
  • This workshop is not focused on architecture for your particular problem space.
  • Most of the things we learn will be only one of many different ways to accomplish a task. I will mention alternatives along the way, but will only focus on a few techniques.
  • There will be coding!
